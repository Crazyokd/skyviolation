# skyviolation
下载番剧——《天空侵犯》到本地。

## 功能简介：
下载《天空侵犯》到本地。（朋友想看，无意中用这种方式给做出来了:flushed:）

## 使用方法：
### 1. 克隆仓库
```shell
git clone https://github.com/Crazyokd/skyviolation
```
### 2. 安装依赖：
```shell
pip install -r requirements.txt		#无需重复安装
```
### 3. 运行脚本：
```shell
python main.py
```

> 番剧将会下载在[sky_violation](sky_violation/)文件夹中，请放心食用:coffee:
